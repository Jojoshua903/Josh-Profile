# Josh-Profile

Source for [joshibrahim.nl](https://joshibrahim.nl) — my personal portfolio site, plus a
collection of browser games I've built along the way. All hand-written HTML, CSS, and
JavaScript, deployed via GitHub Pages.

## About

I'm Josh Ibrahim, a pre-university (VWO) student at Alfrink College in the Netherlands.
I spend my free time coding, building websites, and automating things on GitHub. This repo
is where my portfolio and my experiments live.

## The site

The portfolio (`index.html`) is a single-page site built from scratch, featuring:

- **Custom particle cursor** — an interactive "antigravity" ring-particle field that reacts to the mouse
- **GSAP animations** — staggered hero reveals, scroll-triggered section animations, and count-up stats
- **Bilingual support** — English / Dutch toggle powered by `lang.js` (preference saved to `localStorage`)
- **Photo gallery** (`photos.html`) — with a hidden admin panel (press `Shift + A`) for adding Imgur images, synced via Firebase
- **Responsive + print-friendly** layout
